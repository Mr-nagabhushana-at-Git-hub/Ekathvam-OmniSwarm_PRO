# UI Rules — Ekathvam-OmniSwarm (OmniSwarm by Ekathvam) — a dual/"Twin Engine" multi-agent orchestrator

- Use the tokens in `ui-tokens.md` — never hard-code colors/spacing.
- Mobile-first, responsive; no horizontal scroll on the page body.
- WCAG AA: contrast ≥ 4.5:1, focus states visible, all interactive elements keyboard-reachable.
- Loading, empty, and error states for every async view.
- Consistent component API; document each component in `ui-registry.md`.
- Motion is purposeful and ≤ 250ms; respect `prefers-reduced-motion`.
